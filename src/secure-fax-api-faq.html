  • What is a Fax API?
    An Application Programming Interface (API) is a set of routines, protocols, and tools for building software applications an API specifies how software components should interact. Additionally APIs are used when programming graphical user inteface (GUI) components. [wikipedia]
  • What can we do with an FAX API?
    An API can be “Consumed” by any number of websites, tools and systems. If you have an EMR or CRM you can integrate with our API to have a fax automatically send whenever you add a new customer or any other functionality. Your IT staff can integrate with our API and send/receive faxes from any application that supports custom integrations.
  • In what languages is your FAX API available?
    Our API is written in a format called “REST”. This is an industry standard interface that most API’s comply with. We have samples in PHP, .NET, C#, Python and more. If you use Postman you can check out our custom Postman integration and generate code in dozens of other languages. Any language that can consume an API is compatible with our API.

    We utilize the highest level of TLS encryption available for data in transit either through our secure website or secure API interfaces. We support TLS protected SMTP email with optional REQUIRE TLS extension in accordance with the IETF RFC 3207.
